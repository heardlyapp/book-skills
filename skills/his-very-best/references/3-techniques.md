# 3 — Techniques

> Source: *His Very Best* by Jonathan Alter.

## 5 Methods

### 1. The Rickover Question

Before every decision, ask: "Have I done my best? Is there one more thing I could know, one more call I could make?"

### 2. The Shuttle Negotiation

At Camp David, Carter didn't convene. He shuttled. Walk between cabins. Hear each side privately. Carry offers. Never give up. Saved the Accords 3 times.

### 3. The "I Did Not Lie" Audit

Carter's campaign pledge was unique: he promised never to lie. He kept it. Auditable integrity.

### 4. The Rebuild After Defeat

Carter lost the presidency, owed $1M. He wrote books, built the Carter Center, worked with Habitat. The formula: don't return to power — create new purpose.

### 5. The Small Town Return

Carter went home to Plains. Rootedness is a strategy.
