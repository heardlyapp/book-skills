# 1 — Core Framework

> Source: *Concise Guide to APA Style (7th ed.)* — Chapters 1, 8-10.

## The Two Frameworks

### Framework A: Paper Elements (Chapter 1)

APA Style standardizes the elements of a student paper: Title page (title, author, affiliation, course, instructor, due date, page number) → Text → Reference list → Tables/Figures → Appendices. Running head and abstract are not required for student papers unless requested.

> **Example: Student Title Page** (Chapter 1, Section 1.6): The title is bold, centered, in upper half of page: "Nurturing the Nurses: Reducing Compassion Fatigue Through Resilience Training." One double-spaced line below: "Oliver B. Lee." Below that: "Department of Family and Community Health, University of Pennsylvania." Then course, instructor, due date. Page number flush right in header. Every element has a specific location.

### Framework B: Reference Elements (Chapters 9-10)

The four reference elements are Author, Date, Title, and Source. Format varies by type: journal article, book, chapter, website, etc. DOIs are hyperlinks. Up to 20 authors are included before using ellipsis.

> **Example: Journal Article Reference** (Chapter 10): "Author, A. A., & Author, B. B. (Year). Title of the article in sentence case. *Title of the Journal in Title Case and Italics, Volume*(Issue), Page–Page. https://doi.org/xxxx"
