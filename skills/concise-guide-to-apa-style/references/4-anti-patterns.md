# 4 — Anti-Patterns

> Source: *Concise Guide to APA Style (7th ed.)* — Common errors.

## 5 Common Mistakes

### 1. Treating APA as a Template

"APA is not a set of templates — it's a comprehensive style." Don't just copy a format. Check the 7th edition rules.

> **Example: The "et al." Change** (Chapter 8): In APA 6, you listed up to 5 authors the first time and then used "et al." In APA 7, you always use "et al." for 3+ authors (unless ambiguity would result). Many papers still use the old rule.

> **Example: The DOI Label** (Chapter 9): APA 6th used "doi:10.1037/0000099-0". APA 7th uses "https://doi.org/10.1037/0000099-0" — no label, just the hyperlink. Many references still show the old format.

### 2. Forgetting DOIs

APA 7 requires DOIs for all works that have them.

### 3. Running Head on Student Papers

Student papers do not require a running head.

### 4. Single Space After Period

APA 7 says one space. Check instructor preference.

### 5. Over-Quoting

"Quoting should be done sparingly." Paraphrasing is preferred.
