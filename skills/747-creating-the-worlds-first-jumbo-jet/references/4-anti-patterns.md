# Mistakes and Missteps — Anti-Patterns

## Anti-Pattern 1: The Schedule Optimism

The 747 program was chronically behind schedule. The original timeline was wildly optimistic.

**Case: The Delivery Delays.** Pan Am expected the first 747 in late 1969. The actual delivery was delayed by months. The engine problems, the wing redesign, and the weight issues all contributed.

**Case: The Learning Curve.** Boeing had never built an airplane this large. Every system was new. The learning curve was steep.

## Anti-Pattern 2: The Engine Dependency

Boeing bet the entire 747 program on a single engine that did not exist yet.

**Case: The JT9D.** The Pratt & Whitney JT9D was the first high-bypass turbofan engine. It had never been built. It had never been tested. The 747 could not fly without it. When the engine ran into problems, the entire program was at risk.

**Case: The Backup Plan.** Boeing had no backup engine. There was no other engine that could power the 747. It was the JT9D or nothing.

## Anti-Pattern 3: The Weight Creep

Every system on the 747 got heavier as the design matured. The cumulative effect was significant.

**Case: The Structural Weight.** The initial weight estimates were too low. As the design was refined, the structure got heavier. Each additional pound required more fuel, which required more structure, which added more weight.

**Case: The Toilet Problem.** A small example with big consequences: the original 747 lavatory design was too heavy. Sutter's team had to redesign it to save weight. Every pound mattered.
