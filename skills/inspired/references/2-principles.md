## Empowered vs Feature Teams

**Feature Teams** (the old way): Given a spec → implement it → ship it → get next spec. No ownership of outcomes.

**Empowered Teams** (the new way): Given a business problem to solve → discover the solution → build it → measure results → iterate.

**Key difference:** Empowered teams own the outcome. Feature teams own the output.

## The Product Trio
The three core roles working together daily: Product Manager, Product Designer, Tech Lead. Decisions should be made by this trio, not by executives.
