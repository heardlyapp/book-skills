# Reference: The Thousand Brains Theory

## Overview
Hawkins's newer theory extends the original framework. The neocortex contains thousands of cortical columns, each learning its own model of the world.

## Case: Cortical Columns as Independent Thinkers
Each cortical column is a learning machine. It receives sensory input and builds a model of the object or concept it is learning. When you see a cup, thousands of columns model the cup simultaneously from slightly different reference frames.

## Case: Voting for Consensus
Since each column has its own model, how does the brain decide on a unified perception? Columns vote. The model that has the most evidence wins. This voting mechanism is robust — even if some columns learn an incorrect model, the majority decides.

## Case: Reference Frames
Columns use reference frames — coordinate systems — to organize knowledge. The brain uses the same mechanism for physical space (where an object is relative to your body) and for abstract concepts (the structure of a symphony). Reference frames are the fundamental unit of knowledge.

## Case: Movement and Model-Building
Models are built through movement. As you move your eyes, head, or body, the sensory input changes. The column learns that different inputs from different positions all belong to the same object. Without movement, the column cannot build a complete model. This is why babies move constantly — they are building models of the world.

## Case: The Model of Everything
The thousand brains theory suggests that the entire neocortex is modeling everything simultaneously. Each column models a small piece of the world. The sum of all column models is your understanding of reality. There is no single representation of a cup — there are thousands of representations, each from a slightly different perspective.
