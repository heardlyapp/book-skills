# 反模式：疏离生命的语言（豺狗语言）

> 源自《非暴力沟通》第 2 章 "Communication That Blocks Compassion"

这是 NVC 要替换掉的总靶。**识别它们是把信息翻译成四要素的第一步。** 它们让人疏远自己和他人的需要、引发暴力。马歇尔认为其根植于性恶论与等级制社会。

---

## 五大主形态

### 1. 道德评判
当他人行为与我们价值观不符，就认为他是错的/坏的。
- 形式：「指责、侮辱、贴标签、批评、比较、分析」。
- 例：「你太自私了」/「她很懒」/「他们有偏见」。
- 实质：评判是在用**悲剧性的方式**表达自己的价值观和需要，引发对方防卫抗拒。
- 注意区分：**价值判断**（珍视诚实/和平等品质）≠ **道德评判**（指责不符者）。「暴力是不好的」可改为「对于通过暴力解决冲突，我很担心，希望选择其他方式」。

### 2. 做比较
拿自己/他人与某标准比较，切断对人对己的善意。
- 「一旦我们开始比较，就会感到郁闷！」（《如何让自己活得很悲惨》：与模特身材比、与莫扎特成就比）。

### 3. 推卸责任（否认选择）
把行为或感受的责任归于外部，否认每个人都为自己的思想/情感/行为负责。
- "不得不"式：「有些事不管你喜不喜欢都不得不做」。
- "让人感到"式：「你让我感到内疚」。
- **艾希曼"军官语言"**：把行为归于上级命令/公司政策/法律规定。
- 归咎对象：模糊外部因素 / 个人状况诊断 / 他人行为 / 权威命令 / 群体压力 / 制度规定 / 角色 / 无法抑制的冲动。
- 解药：把"我不得不"改成"我选择…因为我想要…"（见 empathy-and-self.md）。

### 4. 强人所难的要求
以"要求"表达诉求：不配合就（明或暗）指责或惩罚。
- 「在听见要求时，一个人通常只能看到两种选择：屈服或者反抗。」
- 检验：看对方拒绝后你是否指责他（指责=要求；同理=请求）。

### 5. 奖惩思维 / "应该""活该"
相信有的行为该奖、有的该罚；"活该""他该受罚"。
- 「'应该'…它充满暴力…暗示别无选择」— 人天生抗拒被"应该"支配（不论来自外部还是内在暴君）。
- 隐含假设：做错事的人是"坏人"，应受罚悔改。但「所有人都渴望改变，是因为明白改变能带来益处，而不是因为不想受罚」。

## 沟通/同理场景的衍生反模式

- **伪感受**：`我感到 + 你/他/它/像/好似`，或 `我感到被X`（被忽视/误解/冷落/抛弃/利用…）— 是评判或想法，不是感受。
- **用安慰/建议代替同理**：见 empathy-and-self.md 的"妨碍同理的 11 种行为"。急着搞定问题反而阻断连结。
- **机械套用四要素**：走流程而无连结意图，或借四要素之名行改变他人之实 — 「机械地复述…而不清晰这样做目的何在」。

## 冲突/管教场景的衍生反模式

- **惩罚性强制力**：打骂/羞辱/剥夺，让人吃苦悔改 → 引发仇恨、抗拒，削弱善意与自尊。
- **妥协式冲突解决**：各退一步、无人真正满意（NVC 要的是"满意"不是"妥协"）。
- **分析/诊断对方**：「你没有责任感」「你无动于衷」— 会被听成批评，升级冲突。

## 赞赏场景的衍生反模式

- **评判式赞美 / 用表扬操控**：正向评判也是疏离生命的沟通。「一旦人们发现表扬是为了控制，他们的生产力就会下滑。」用感激三要素替代（见 voice-and-gratitude.md）。
- **故作谦虚 / 自负地接收感激**：接收赞赏的两种极端，都断了连结。

## NVC in Action 案例参考

### 案例：道德评判如何升级冲突（第 2 章）
一位母亲对青春期的儿子说：「你真的很自私!」
她以为在表达问题，实际是道德评判 → 儿子愤怒反击 →
双方陷入"指责-反击"的死循环。

NVC 改写后：「你上周五晚上没打电话，我有些担心，因为我希望我们之间保持连结。你愿意告诉我你在忙什么吗?」

**关键**：把"自私"这个标签，翻译回具体的观察+"我"的感受+需要+请求。

### 案例：推卸责任的代价（第 2 章）
执行纳粹屠杀的艾希曼在受审时说：「我只是在执行命令。」马歇尔指出，这是"否认选择"的极端案例 — 把行为的责任转嫁给外部权威。

**在日常生活**中，说"我不得不加班"等于否认了自己的选择权。解药是把"不得不"改成"我选择…因为…"
